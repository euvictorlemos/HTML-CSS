Listas

Listas Ordenadas

listas OL são listas de passo a passo e fazemos ela com a tag < OL >para criar o formato de lista e a tag <LI> para identificar cada item da lista, veja exemplo:

  1. item
  2. item
  3. item
  4. item

Você pode mudar as numerações da lista por outras especificações colocando a TAG TYPE dentro da TAG <ol> veja o exemplo:

  1. item
  2. item
  3. item
  1. item
  2. item

Listas Não Ordenadas

Lista não ordenada, são as listas que não necessitam de uma ordem para especificar cada coisa

Para fazer uma lista não ordenada basta usar a tag <ul> e depois usar a tag <li> veja o exemplo:

  • item
  • item
  • item

Para mudar o formato de disc da lista para outro formato basta escrever type como na lista anterior, veja:

  • item
  • item
  • item
  • item

Misturando Listas

Também podemos misturar as listas, veja:

  1. item
  2. item
    1. item 2.1
    2. item 2.2
  3. item
    • item 3.1
      • bônus
  4. item

Listas de Definição

São listas como um dicionário, e para faze-las precisamos por a Tag <DL> a Tag <DT> (o termo/nome) e por último a Tag <DD> (explicaçao ou descrição) veja os exemplos:

HTML5
Linguagem de Marcação utilizada para criar conteúdo dos sites
CSS3
Linguagem de Marcação utilizada para criar estilos de sites
JavaScript
Linguagemde marcação utilizada para criar interativos de sites
